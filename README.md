#AKA
aka is your alias's best friend. Add, edit, remove and manage aliases from terminal.

![demo](gif/demo.gif)


#### how to install
```
curl -o /usr/local https://raw.githubusercontent.com/ytbryan/aka/master/aka 
#or
sudo cp aka /usr/local/
```
